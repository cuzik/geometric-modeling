# [WIP] Geometric Modeling

## Requiries

First install some libs "(>_<)"

### Ubuntu/Linux

- install pygame

```sh
sudo apt install python-pygame
```

- install pip

```sh
sudo apt install python-pip
```

- install scipy

```sh
pip install scipy
```

- install numpy

```sh
pip install numpy
```

- isntall tkinter

```sh
sudo apt install python-tk
```
